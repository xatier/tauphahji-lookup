# 鬥拍字 Kóng-uē

Chrome extension for looking up `selected text` on [鬥拍字.意傳.台灣](https://xn--p8s96olm5c.xn--v0qr21b.xn--kpry57d).


## Installation

1. Clone the repo to wherever-you-like
2. Navigate to `chrome://extensions` on Chromium.
3. Load unpacked
